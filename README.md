# TsunamiShield AI Dashboard

A modern, real-time tsunami disaster management simulation for Chennai, India.

## Key Features

1.  **Interactive Map (Leaflet.js)**:
    *   Centered on the Chennai coast with dark-mode styling.
    *   Visualizes **Risk Zones** (Red: High, Yellow: Moderate) and **Safe Zones** (Green).
    *   Dynamic evacuation route overlay when simulation starts.
2.  **Simulation Control Panel**:
    *   Adjustable wave height and selection of coastal areas.
    *   Count-down timer to impact.
    *   "Start Simulation" button triggers localized emergency protocols.
3.  **Real-Time Alerts**:
    *   Animated "EMERGENCY" status indicators and banners.
    *   Active alert boxes with critical evacuation instructions.
    *   Simulation-triggered siren sound effects.
4.  **Integrated Evacuation Module**:
    *   Automated routing from danger zones to the nearest safe ground.
    *   Real-time statistics on population at risk and estimated evacuation times.
5.  **Multi-Channel Simulation**:
    *   Live notification feed for internal systems.
    *   SMS Broadcast preview panel for citizen alerts.
6.  **Responsive UI**:
    *   Premium dark-theme design with glassmorphism and subtle gradients.
    *   Fully functional on desktop and mobile viewports.

## Tech Stack
*   **HTML5/CSS3**: Custom dark theme with blurred glass effects.
*   **JavaScript**: Vanilla logic for simulation and UI reactivity.
*   **Leaflet.js**: High-performance mapping and geospatial visualization.
*   **Lucide Icons**: Crisp, vector-based dashboard iconography.

## How to use
1.  Open `index.html` in your browser.
2.  Use the **Simulation Panel** on the left to select a wave height.
3.  Click **Start Simulation** to trigger the alarm, map updates, and evacuation routing.
4.  Monitor the **Live Notifications** and **SMS Preview** for system progress.
